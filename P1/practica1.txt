IMPLEMENTACIÓN DEL PROGRAMA:

Primero, definimos todas las variables que necesitaremos,
incluyendo una para cada contador específico y otra para
llevar el total de palabras del texto. Dentro de cada regla,
identificamos las palabras o frases correspondientes,
actualizamos su contador, incrementamos el número total
de palabras y sumamos la cantidad de caracteres usando la
variable yyleng, que FLEX proporciona automáticamente. En
resumen, creamos una regla para cada tipo de palabra que
queremos reconocer. Al final, imprimimos el valor de cada
variable para mostrar los resultados.

En esta práctica se han añadido símbolos especiales como la ñ y otros
para una mayor completitudo del lenguaje, además de haber añadido una
regla que estipula casi todos los números como se pueden representar.
